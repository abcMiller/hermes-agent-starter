# Memory 与 Skills

## 1. Memory 是事实记忆

Memory 用于记录长期事实：

- 用户偏好
- 项目背景
- 环境信息
- 常用约定

文件：

```text
data/memories/USER.md
data/memories/MEMORY.md
```

## 2. Skills 是过程记忆

Skills 用于记录“怎么做”：

- 如何设计页面
- 如何审查代码
- 如何写文档
- 如何做 Playwright 自动化

文件：

```text
data/skills/<skill-id>/SKILL.md
```

## 3. Memory 与 Skill 的区别

| 类型 | 记录内容 | 示例 |
|---|---|---|
| Memory | 事实 | 用户喜欢中文、项目用 TypeScript |
| Skill | 流程 | 如何写 README、如何做代码审查 |

## 4. 后续改进

- Memory 写入审核
- Skill 自动生成
- Skill 版本管理
- Skill 命中评分
- 历史会话检索增强
